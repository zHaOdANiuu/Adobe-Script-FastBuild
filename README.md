# Adobe-Script-FastBuild

estk+rolldown

- 目前只支持ae模板, 后续会支持其他模板

## install
npm
```
npm i -g estkbuild
```
pnpm
```
pnpm add -g estkbuild
```

## run
```
npx estkbuild create youDirName -t ae && cd youDirName && npm install && npm run build
```
